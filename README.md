Cara eksport database yang ada di mysql dengan bantuan third party FPDF
1. Download FPDF terlebih dahulu di fpdf.org
2. extract lalu masukkan folder ke dalam project
3. hubungkan project dengan file fpdf terutama fpdf.php atau juga bisa dengan syntax require 'fpdf.php' jika di dalam tingkat folder yang sama
4. setelah berhasil dihubungkan, maka funtion fpdf siap digunakan.
